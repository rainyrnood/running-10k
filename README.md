# running-10k

10K 5:15/km 훈련 로그 — 2026.09.12 레이스 대비 15주 커리큘럼 + 일일 체크/기록 앱.

- `index.html` — 날짜별 일일 체크/기록 앱 (iPhone 세로 최적화, localStorage 저장, JSON 백업)
- `plan.html` — 전체 커리큘럼 문서 (참고용)

데이터는 브라우저 localStorage에만 저장되며, 백업은 JSON 내보내기/가져오기로 합니다.
